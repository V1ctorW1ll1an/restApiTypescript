Restful API example

# Setup

## Docker Compose setup

```bash
$ docker compose up -d
$ docker compose exec db bash
$ psql -h 0.0.0.0 -U postgres
$ CREATE DATABASE <db-name>
```

## Run server

```bash
$ pnpm install
$ pnpm run dev
```

## Install packages

```bash
$ pnpm install <package-name>
```
